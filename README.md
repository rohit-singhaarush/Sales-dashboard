# 📊 Sales Dashboard Project (Excel)

## 📌 Project Overview
This project analyzes sales data using Microsoft Excel to generate meaningful insights and build an interactive dashboard.

The project focuses on transforming raw data into a structured format, performing data cleaning, and creating visualizations that help in understanding business performance.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Track profit and revenue trends  
- Identify top-performing products  
- Understand the impact of discounts on sales  
- Monitor monthly and category-wise performance  

---

## 🛠 Tools & Technologies Used
- Microsoft Excel  
- Pivot Tables  
- Charts & Graphs  
- Data Cleaning Techniques  
- Basic Excel Formulas  

---

## 📂 Dataset Information
The dataset includes the following columns:
- Sale Date  
- Store Location  
- Product Category  
- Product Name  
- Quantity  
- Unit Price  
- Discount  
- Total Sales  
- Profit  

---

## 📈 Key Features
- Interactive Sales Dashboard  
- Monthly Sales Trend Analysis  
- Category-wise Performance  
- Top Products Identification  
- Profit & Discount Analysis
